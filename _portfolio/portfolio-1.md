## Project Overview
This project presents a comprehensive pipeline for simulating and analyzing autonomous driving behaviors using reinforcement learning methods. It integrates custom-designed driving environments, multi-agent interactions, trajectory visualization, and performance analytics to support research on robustness, generalization, and driving style interpretation.

## Key Contributions
- **Environment Design:** Built realistic driving scenarios (e.g., lane-keeping, merging, roundabout) with adjustable traffic density, sensor settings, and vehicle dynamics.
- **Reinforcement Learning Integration:** Implemented PPO, RGDR, and risk-aware RL algorithms to evaluate robustness under various domain randomization strategies.
- **Trajectory Visualization:** Developed tools for real-time and offline visualization, including ego-vehicle trajectories, reference paths, dynamic obstacles, and prediction overlays.
- **Performance Evaluation:** Analyzed reward curves, safety metrics, policy stability, and out-of-distribution performance across multiple environments.

## Technical Highlights
- Python-based simulation with full logging and replay functionality  
- Custom evaluation metrics for uncertainty handling and risk-sensitive behavior  
- Adaptive visualization dashboards for comparing multiple policy runs  
- Modular design enabling rapid scenario creation and policy benchmarking  

## Outcome
This portfolio project serves as a foundational tool for analyzing autonomous driving policies and has been used in subsequent research on domain randomization, driving-style modeling, and robust decision-making.
